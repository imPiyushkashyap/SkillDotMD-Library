# skill-registry

Open registry of AI agent skills for Claude Code, Cursor, Windsurf, Cline, GitHub Copilot and more.

## Install any skill with one command

```bash
npx skills add anthropics/skills/pdf
npx skills add stripe/ai/stripe-best-practices
npx skills add vercel-labs/agent-skills/vercel-react-best-practices
```

## What is this

This registry is the backbone of [SkillDotMD](https://skilldotmd.com) — a tool that lets you browse, generate, and install AI agent skills in one command.

It contains two things:

- `index.json` — a curated index of the best public skills from official teams like Anthropic, Vercel, Stripe, Cloudflare, Supabase, Expo, Better Auth, and Neon
- `generated/` — skills generated by the SkillDotMD AI generator, installable via `npx skills add`

## Browse skills

Visit [skilldotmd.com/browse](https://skilldotmd.com/browse) to search and filter all skills.

## Generate a skill

If a skill doesn't exist yet, go to [skilldotmd.com](https://skilldotmd.com), describe the library or paste a docs URL, and get a ready-to-install skill in seconds.
